Offline Inventory Control System with Real-Time Stock Monitoring and Receipt Generation

Grocery Tracker is an Android-based inventory management application designed for small grocery stores and shop owners. 
The app works both offline and online, allowing vendors to efficiently track stock levels, manage products, and generate receipts instantly.
